🎙️ Voice Assistant Guide

How to Use

The assistant is always listening. Just speak:

  • "help" / "मदद" / "సహాయం" → Emergency alert
  • "go detection" / "डिटेक्शन जाओ" / "డిటెక్షన్ కి వెళ్ళండి" → Open camera
  • "go home" / "घर जाओ" / "ఇంటికి వెళ్ళండి" → Navigate home
  • "start camera" / "कैमरा चालू" / "కెమెరా స్టార్ట్" → Start detection
  • "stop camera" / "कैमरा बंद" / "కెమెరా స్టాప్" → Stop detection
  • "flip camera" / "कैमरा बदलो" / "కెమెరా ఫ్లిప్" → Switch camera
  • "sound on/off" / "आवाज़ चालू/बंद" / "సౌండ్ ఆన్/ఆఫ్" → Toggle voice
  • "switch to Hindi / తెలుగు" → Change language
